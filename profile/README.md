# ZapretDesktop

Организация проекта **ZapretDesktop** — десктопного приложения для Windows, которое управляет обходом DPI (глубокой инспекции пакетов) на базе [zapret](https://github.com/bol-van/zapret) / [Flowseal](https://github.com/Flowseal/zapret-discord-youtube).

## Репозитории

- **[ZapretDesktop](https://github.com/ZapretDesktop/ZapretDesktop)** — исходный код приложения (Python, PyQt6). Сборка в один исполняемый файл для Windows 10/11 (x64).
